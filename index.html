🏆 主要な学生落語大会

学生落語家たちが目標にする、全国規模の代表的な大会一覧。高座のブラッシュアップや目標設定の参考にしよう。

■ 策伝大賞（全日本学生落語選手権）

岐阜県岐阜市で毎年開催される、学生落語の甲子園とも呼ばれる最大規模の大会。全国から数多くの学生が集まり、落語の祖とされる「策伝上人」にちなんで日本一を競い合う。

■ 桂福丸杯（全国学生落語いなみ大会）

兵庫県稲美町で開催される大会。プロの落語家である桂福丸師匠がプロデュースに携わっており、学生らしい瑞々しさと技術を競い合う熱い高座が繰り広げられる。

■ ちりとてちん杯（全国女性学生落語大会）

福井県小浜市で開催される、全国の「女性学生落語家」を対象としたユニークな大会。NHK連続テレビ小説『ちりとてちん』の舞台となった地で、華やかかつハイレベルな戦いが行われる。

📖 基礎知識

落語研究会 必須用語集

落研の活動や寄席の番組（プログラム）を理解する上で必要な専門用語。

📌 番組（香盤・出番）に関する用語
開口一番（かいこういちばん）

落語会や寄席の一番最初、トップバッターの枠のこと。基本的には前座（落研では新人や後輩）が担当し、客席を温める役割を持つ。

中トリ / 仲トリ（なかとり）

前半戦の最後、すなわち「仲入り」の休憩前に入る最も格の高い出番のこと。前半戦の主役として大きな盛り上がりを作る役割を担う。

仲入り（なかいり）

寄席や長時間の落語会における中休みのこと。中トリの出番が終わったタイミングで設けられる。

シバリ

仲入りの休憩が明けた直後、後半戦の一番最初（食いつき）の出番のこと。休憩気分の客席を再び落語の世界に引き戻す技量が求められる。

モタレ

最後の主役（トリ）のすぐ前に登場する出番のこと。トリの演者の邪魔をしないよう配慮しつつ、客席を最高潮に温めてバトンを渡す重要なポジションである。

主任 / トリ（しゅにん / とり）

その日の落語会や寄席の最後を締めくくる、一番格上の演者のこと。興行全体の責任を背負って高座に上がる。

🎪 会（公演）の種類に関する用語
独演会（どくえんかい）

特定の落語家が単独で主催し、メインとして複数のネタを披露する落語会のこと。演者個人の実力や世界観を深く堪能できる公演形式である。

二人会（ににんかい／ふたりかい）

2人の落語家が並び立ってメインを務める落語会のこと。実力派同士の対決や師弟対談など、2人のネタの対比や掛け合いを楽しむ。

寄席（よせ）

年中無休で毎日、落語や色物（漫才やマジック）を交代で上演している常設の劇場のこと（東京の鈴本演芸場、新宿末廣亭、大阪の天満天神繁昌亭など）。多くの演者が10〜15分程度の短いネタで次々と登場する。

落語会（らくごかい / ホール落語）

市民ホールや劇場を借りて、特定の主催者が企画する単発の落語公演のこと。出演者やネタがあらかじめ決まっていることが多い。

勉強会（べんきょうかい）

落語家が新ネタに挑戦したり、稽古の成果を試したりするために、少人数の前で低価格で行う実験的な落語会のこと。落研における内々の練習発表の場としても用いられる。

🏯 その他落語の基本用語
前座 / 前座噺（ぜんざ / ぜんざばなし）

開口一番に演じられる、短くて分かりやすい滑稽噺のこと。落語の基本が凝縮されており、初心者が最初に覚えるべきネタである。

サゲ / オチ

噺の結末、最後に落とす部分のこと。「地口サゲ（駄洒落）」「仕込みサゲ」「逆さサゲ」など様々な種類が存在する。

出囃子（でばやし）

落語家が高座に登場する際に、三味線や太鼓で演奏されるテーマ曲のこと。演者ごとに専用の曲が決まっている。

落語研究会 必須用語集

落研の活動や寄席の番組（プログラム）を理解する上で、絶対に覚えておきたい専門用語です。

📌 番組（香盤・出番）に関する用語
開口一番（かいこういちばん）

落語会や寄席の一番最初、トップバッターの枠のこと。基本的には前座（落研では新人や後輩）が担当し、場を温める役割を持ちます。

中トリ / 仲トリ（なかとり）

前半戦の最後（仲入り休憩の前）に登場する、非常に重要なポジションの演者。ここで一度大きな盛り上がりを作ります。

仲入り（なかいり）

寄席や長時間の落語会における「休憩時間」のこと。中トリの出番が終わったタイミングで入ります。

シバリ

仲入りの休憩が明けた直後、後半戦の一番最初（食いつき）の出番のこと。休憩気分の客席を再び落語に引き戻す技量が求められます。

モタレ

最後の主役（トリ）のすぐ前に登場する出番のこと。トリの演者が得意なネタや雰囲気を邪魔しないよう配慮しつつ、客席を最高潮に温めてバトンを渡す重要な役割です。

主任 / トリ（しゅにん / とり）

その日の落語会や寄席の「大トリ」、つまり最後の主役となる一番格上の演者です。その会全体の責任を背負って高座に上がります。

🎪 会（公演）の種類に関する用語
独演会（どくえんかい）

特定の落語家が1人で主催し、メインで何席も（2〜3席ほど）落語を披露する会。その人の実力や世界観をたっぷり堪能できる会です。

二人会（ふたりかい）

2人の落語家が並び立ってメインを務める落語会。ライバル同士や仲の良い師匠同士など、2人の掛け合いやネタの対比を楽しみます。

寄席（よせ）

年中無休で毎日落語や色物（漫才やマジック）を交代で上演している常設の劇場のこと（鈴本演芸場や末廣亭など）。多くの演者が10〜15分程度の短いネタで次々と登場します。

落語会（らくごかい / ホール落語）

市民ホールや劇場などを借りて、特定の主催者が企画する単発の落語公演。出演者やネタがあらかじめ決まっていることが多いです。

勉強会（べんきょうかい）

落語家がまだ本番で掛けたことのない新しいネタ（新ネタ）に挑戦したり、稽古の成果を試したりするために、少人数の前で低価格で行う実験的な落語会。落研でも内々の練習発表の場としてよく使われます。

🏯 その他落語の基本用語
前座 / 前座噺（ぜんざ / ぜんざばなし）

開口一番に演じられる、短くて分かりやすい滑稽噺のこと。基本が凝縮されており、初心者が最初に覚えるべきネタ。

サゲ / オチ

噺の結末、最後に落とす部分のこと。「地口サゲ（駄洒落）」「仕込みサゲ」「逆さサゲ」など様々な種類がある。

出囃子（でばやし）

落語家が高座（ステージ）に登場する際に、三味線や太鼓で演奏されるテーマ曲のこと。演者によって曲が決まっている。
